# ⛽ Fuel Station Viewer (React App)

![Demo Gif](https://i.imgur.com/PORA6Kf.gif)

## 🚀 Project Overview
Fuel Station Viewer is an interactive React-based web application that visualizes fuel stations on a map with dynamic UI interactions.

This project demonstrates:
- Map-based visualization
- Interactive UI components
- Integration with external APIs (Mapbox)

---

## ✨ Features
- 📍 Interactive Map UI
- 🔎 Fuel Station Visualization
- ⚡ Smooth User Experience
- 🗺️ Mapbox Integration

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- CSS3
- Mapbox API
